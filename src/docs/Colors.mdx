# Цвета библиотеки

Все цвета задаются через CSS-переменные. Это позволяет легко менять темы.

## Переменные

- `--color-primary`: Основной цвет кнопок (по умолчанию `#007bff`)
- `--color-secondary`: Вторичный цвет (по умолчанию `#6c757d`)
- `--color-text`: Цвет текста (по умолчанию `#333`)
- `--color-border`: Цвет границ (по умолчанию `#ddd`)
- `--color-background-disabled`: Фон для disabled состояния (по умолчанию `#f5f5f5`)
